# NativeScript Contribution Guide

Thank you for considering contributing to NativeScript! This guide will help you get started.

## Getting Started

1. Fork the repository
2. Create a new branch for your feature
3. Make your changes
4. Submit a pull request

## Code Style

- Use TypeScript for new code
- Follow the existing code patterns
- Add tests for new features

## Testing

Before submitting a PR, make sure:
- All tests pass
- Your code is properly formatted
- Documentation is updated

## Questions?

Feel free to open an issue if you have questions!
